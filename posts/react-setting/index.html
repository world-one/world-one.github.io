리액트 세팅해보기( without CRA )

react.js로 프로젝트를 해본적은 없다.
처음에 vue.js로 작업을 진행했을 때는 신기하고 너무 편했다.
이후로 react에 대한 관심도 많아졌다. vue cli 처럼 cra(create-react-app)라는 보일러플레이트가 있다.
vue cli는 참 편하게 사용했으니 CRA를 이용해서 한번 해볼까 하다가
삽질일지도 모르겠지만 그냥 아무것도 없는 상태에서 세팅을 해보자라는 생각을 했다.
저번달에 이미 해봤지만 기억이 잘 안나는 관계로 다시 진행해보려한다.

npm init

npm init -y

명령어를 입력하면 package.json이 생성된다.

webpack

npm install --save-dev webpack webpack-dev-server webpack-cli
or
yarn add -D webpack webpack-dev-server webpack-cli

webpack.config.js 파일을 만들어주고 아래 내용을 추가한다.

const path = require("path");
const HtmlWebpackPlugin = require("html-webpack-plugin");

module.exports = {
  mode: "development",

  entry: "./src/index",

  resolve: {
    extensions: [".js", ".jsx"]
  },

  module: {
    rules: [
      {
        test: /\.jsx?$/,
        loader: "babel-loader",
        exclude: /node_modules/
      }
    ]
  },
  output: {
    path: path.resolve(__dirname, "dist"),
    filename: "bundle.[hash].js"
  },
  devServer: {
    historyApiFallback: true,
    open: false,
    port: 8080,
    host: 'localhost',
    hot: true,
    inline: true
  },
  plugins: [
    new HtmlWebpackPlugin({
      template: `./public/index.html`
    })
  ]
};

Babel

yarn add -D @babel/core @babel/preset-env @babel/preset-react babel-loader

babel.config.js 파일을 만든 뒤 아래 내용을 추가한다.

module.exports = function(api) {
  api.cache(true);

  const presets = ["@babel/preset-env", "@babel/preset-react"];

  const plugins = [];

  return {
    presets,
    plugins
  };
};

react

yarn add react react-dom

public 폴더에 index.html을 만들어준다.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta http-equiv="X-UA-Compatible" content="ie=edge" />
    <title>Document</title>
  </head>
  <body>
    <div id="root"></div>
  </body>
</html>

src 폴더에 index.js를 만든다.

import React from "react";
import ReactDOM from "react-dom";
import App from "./components/App";

ReactDOM.render(<App />, document.getElementById("#root"));

components 폴더에 App.js를 만든다.

import React from "react";

const App = () => {
  return <div>hello world!</div>;
};
export default App;

리액트는 함수나 클래스로 작성한다.
실제로 프로젝트를 진행하면서 정리해보겠다.

스크립트 추가

package.json scripts에 아래 내용을 추가한다.

"build": "webpack",
"watch": "webpack-dev-server --inline --hot"

yarn build
yarn watch

scss

css 보다 편하다.
scss 설정을 해보자! 

yarn add -D node-sass style-loader sass-loader css-loader

webpack.config.js에 설정을 추가한다.

{
    test: /\.(css|scss)$/,
    use: [
        'style-loader',
        'css-loader',
        {
        loader: 'sass-loader',
        options: {
            sassOptions:{
            includePaths: [`${__dirname}/src/scss`],
            },
            prependData: `@import '/utilities/variables';`
        }
        }
    ]
    },

css-loader는 외부에서 css를 import한 경우 에러가 발생하여 추가했다.
각 파일을 해당 컴포넌트에 import하면 문서내 style tag로 들어간다.

yarn add -D mini-css-extract-plugin

별도로 css파일을 만들어주는 플러그인이다.
webpack.config.js에 아래와 같이 추가해주자!

const MiniCssExtractPlugin = require('mini-css-extract-plugin');

{
    test: /\.(css|scss)$/,
    use: [
        MiniCssExtractPlugin.loader,
    ]
},
plugins: [
    new MiniCssExtractPlugin({ filename: '/src/css/app.[hash].css',})
]

postcss-loader 는 나중에 추가해보도록 한다.

clean-webpack-plugin

build를 하게 되면 이전에 묶였던 파일들이 그대로 남아있다.
매번 삭제하기 번거로우니 추가해주도록 하자.
알아서 삭제해준다.

yarn add -D clean-webpack-plugin

const { CleanWebpackPlugin } = require('clean-webpack-plugin');

plugins: [
new CleanWebpackPlugin({
    cleanAfterEveryBuildPatterns: ['dist']
}),

router

vue.js는 공식 라우터가 있다.
react는 서드파티 라이브러리를 사용해야한다.

yarn add react-router-dom

사용 예시

app.js
import { BrowserRouter } from 'react-router-dom';

<div className="wrap">
    <BrowserRouter>
        <Header />
        <Routes />
    </BrowserRouter>
</div>
Header.js
import { Link } from 'react-router-dom';
<header>
    <ul>
        <li>
            <Link to="/">main</Link>
        </li>
        <li>
            <Link to="/about">About</Link>
        </li>
    </ul>
</header>
Routes.js
import { Route, Switch } from "react-router-dom";
<main>
    <Switch>
        <Route exact path="/" component={Main} />
        <Route path="/about" component={About} />
    </Switch>
</main>

BrowserRouter가 감싸고 그 안에 Link로 이동할 주소를 설정한다.
Route, Switch는 해당 주소에서 선택된 component를 보여준다.
